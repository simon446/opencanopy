# mechanical/

Frame, bays, mounts, and printable parts for OpenCanopy. Owned by the **Mechanical** track
(spec §8, §12).

## Layout

- `cad/` — design source.
  - `source/` — **parametric build123d model** (Python); see `cad/source/README.md`
    for toolchain + how to regenerate. This is the dimensional authority.
  - `step/` — neutral STEP exports (full assembly + every module), regenerated by
    `cad/source/build.py`.
- `stl/` — printable geometry, regenerated from source.
  - `printable/` — release-quality, manifold STLs intended for end users.
  - `prototypes/` — tolerance coupons / experimental prints.
- `drawings/` — orthographic assembly views, generated by `cad/source/drawings.py`.
- `cad-verification-checklist.md` — §12.1 checklist, numbers computed by `verify.py`.
- `bom-mechanical.csv` — mechanical bill of materials (pot, reservoir, frame, …).
- `print-settings.md` — material/zone + slicer settings (PETG/ASA/ABS per §16.2).
- `fit-tests.md` — tolerance-coupon results; feeds calibrated clearances to `params.py`.
- `alpha-build.md` — assembly procedure + alpha build log (M5-09).

## Regenerate artifacts

```bash
.venv-cad/bin/python mechanical/cad/source/build.py        # STEP + STL
.venv-cad/bin/python mechanical/cad/source/verify.py       # §12.1 numbers
.venv-cad/bin/python mechanical/cad/source/drawings.py     # SVG views
.venv-cad/bin/python scripts/stl_check.py --dir mechanical/stl   # CI manifold gate
```

## Key constraints

Open-frame appliance within the **locked 480 × 320 × 700 mm envelope** (see
`docs/product-requirements.md`): electronics in an isolated upper dry bay, water in
the bottom wet bay, removable 10 L pot and 4 L reservoir, leak tray, fan guard, and
stainless fasteners near water (spec §8, §16.2, §17). The model asserts the envelope
and the "water fails downward / electronics live upward" zone separation (§6.2) on
every build.
