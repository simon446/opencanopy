## Objective

Write the build guide that takes a maker from printed parts + BOM to a flashed, working unit.

## Deliverables

- [ ] `docs/assembly.md`: step-by-step mechanical assembly with photos/renders.
- [ ] Wiring per [WI-EE-05](../03-electronics/WI-EE-05-harness-pinout.md) harness table.
- [ ] Firmware flashing guide (USB-C/UART), including first-boot self-test expectations.
- [ ] References print-settings and fit-tests from [WI-ME-08](../04-mechanical/WI-ME-08-tolerance-alpha-build.md).

## Acceptance criteria

- Assembly + flashing guide complete (§14.3 / §21).
- A builder can go from parts to a booting unit using only this guide.
