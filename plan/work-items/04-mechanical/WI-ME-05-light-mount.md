# WI-ME-05 — Light mount

| Field | Value |
|---|---|
| Track | Mechanical |
| Milestone | M5-05 |
| Depends on | WI-ME-01 |
| Spec refs | §7.2 (mounting), §8.6 |
| Status | Not started |

## Objective

Design the overhead LED mount with adjustable height, safe clearance, secondary retention, and a
thermal path that does not dump heat into electronics or plastics.

## Deliverables

- [ ] CAD/STL for the light mount; adjustable height preferred (150–300 mm above canopy, §8.6).
- [ ] Mechanical secondary retention so the fixture cannot fall into plant/wet area (§7.2).
- [ ] Driver heat isolated from canopy and plastic parts; LED-adjacent parts in ASA/ABS/heat-proven PETG.
- [ ] If fixed-height, total grow-zone height ≥600 mm (§8.6).

## Acceptance criteria

- Adjustable or fixed mount with verified clearance (spec §15.6 M5-05).
- Secondary retention present; thermal path validated in [WI-QA-03](../05-validation-qa/WI-QA-03-thermal.md).
